# IRSC / CIHR Funding results Dashboard

This dashboard serves as an interactive tool to analyze and explore health research funding trajectories focusing on project grant. 
Link to the application: https://irsc-canda-analysis-app.streamlit.app/

## Tools/libraries used: 
- Streamlit
- Pandas
- altair
- pathlib

# Data sources
- CIHR Provincial Funding Summaries: https://cihr-irsc.gc.ca/e/51315.html
  Provides official provincial-level funding allocations and comparison metrics aggregated by competition.****
- CIHR Funding Decisions Database: https://webapps.cihr-irsc.gc.ca/decisions/p/main.html?lang=en#sort=namesort%20asc&start=0&rows=20
  Provides highly detailed institutional, researcher, and program-level data. The data was extracted with filter on program: Project Grant.
